Данная база данных была взята из репозитория https://github.com/morenoh149/postgresDBSamples.
Для развертывания на локальной машине, необходимо скачать папку "data" из данного репозитория.

Файл install.sql будет выполнен при инициализирующем старте Вашего docker контейнера.
В docker-compose Вы можете увидеть, что данная папка была примонтирована к инстансу postgres.
